# SOP-007 — Slurry Tank Agitator Lockout / Tagout (LOTO)
Revision: 1 | Status: SUPERSEDED BY REV 2 — DO NOT USE
SYNTHETIC DEMO DOCUMENT — generic refinery, not any real plant.

## 1. Purpose
Defines the energy isolation steps required before any work on slurry tank
agitators.

## 2. Locks — OLD RULE (superseded)
2.1 For work expected to last less than one hour, a single lock fitted by
    the shift supervisor on behalf of the work party is sufficient.
2.2 For longer work, each person should fit a personal lock where
    practical.

NOTE: This revision's locking rule was replaced in Rev 2 after an industry
incident review. Rev 2 requires a personal lock from EVERY worker with no
short-duration exception. This Rev 1 document is retained for record only.

## 3. Isolation Steps
3.1 Stop the agitator from the control room.
3.2 Open and tag the motor isolator at the local switchroom.
3.3 Try-start from the local start button to prove de-energisation.

## 4. Restoration
4.1 Supervisor removes the work-party lock when all work is reported
    complete.
